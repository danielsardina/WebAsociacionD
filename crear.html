Crea tu propio videojuego

Como empezar

Para comenzar hay que elegir un motor de videojuegos, o elegir un lenguaje de programacion y una libreria gráfica. Hay principalmente 3 motores de videojuegos que usamos en nuestros juegos, Unity, Unrial engine y Godot. Estos motores son fáciles de utilizar y hay un montón de informacion en internet de como utilizarlos.

Elegir una categoria

Una vez elegido el motor, hay que elegir una categoria. Puedes ver todas las categorias disponibles en la pagina de videojuegos y elegir la que prefieras. Cada juego debe tener un categoria, y todo el contenido debe estar relacionado con esa categoria.

Crear el juego

Para poder crear un juego se necesita una gran cnatidad recursos, imagenes, sonidos, textos... Crear estos recursos lleva mucho tiempo, por lo que es una buena idea buscar recursos ya creados. En Web Asociacion te proporcionamos estos recursos, y si no tenemos lo que necesitas, hay varios sitios en internet, donde se pueden conseguir estos recursos.

Publicar el juego

Una vez terminado el juego, hay que hacer todas la pruebas necesarias, para comprobar que el juego funcione en todas la plataformas para las que a sido diseñado. Después debes subir el juego a nuestra página, para que comprobemos que cumple con todos los requisitos. Una vez terminadas las pruebas, nosotros nos encargaremos de publicar el juego.
